# Лабораторна робота №1
## Створення та налаштування простого сервера зі статичним контентом

### Мета роботи
Ознайомитися з веб-сервером nginx та навчитися налаштовувати статичний сайт.

### Використані технології
- nginx
- HTML, CSS
- Ubuntu (WSL)

### Результат
Налаштовано локальний веб-сервер nginx, який відображає статичну HTML-сторінку на порту 8080.
